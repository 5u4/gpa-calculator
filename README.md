# GPA Calculator

## Description

A Wechat mini program built on [mpvue](http://mpvue.com/) for calculating GPA in 4.00 and 4.33 scales.

## Deployment

1. Build

```bash
$ npm run build
```
2. Deploy

Deploy files in `dist` to server
